# CCNA Labs Repository  
## مستودع لابات CCNA  

### نظرة عامة | Overview
هذا المستودع يحتوي على مجموعة من اللابات التدريبية الخاصة بمنهج CCNA. الهدف من هذه اللابات هو توثيق خطوات الإعداد والتجارب العملية في مجالات مثل:  
- تكوين الشبكات المحلية (LAN)  
- إعداد الـ VLANs  
- بروتوكولات التوجيه (Routing Protocols)  
- التحكم بالوصول (ACLs)  
- استكشاف الأخطاء وإصلاحها (Troubleshooting)  

This repository includes a collection of CCNA training labs. The purpose is to document configuration steps and hands-on experiments in areas such as:  
- Local Area Network (LAN) setup  
- VLAN configuration  
- Routing protocols  
- Access Control Lists (ACLs)  
- Troubleshooting practices  

---

### الهدف | Purpose
الغرض من هذا المستودع هو:  
- توثيق رحلة التعلم الخاصة بي في مجال الشبكات.  
- مشاركة المعرفة مع الآخرين المهتمين بالـ CCNA.  
- إنشاء مرجع يمكن الرجوع إليه لاحقًا.  

The purpose of this repository is to:  
- Document my learning journey in networking.  
- Share knowledge with others interested in CCNA.  
- Build a personal reference for future use.
